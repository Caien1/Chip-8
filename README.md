# CHIP-8 Emulator

A CHIP-8 emulator written in Odin.


## Features

* CHIP-8 CPU implementation
* 4 KB memory
* ROM loading
* Fetch-decode-execute cycle
* Opcode decoding using bitmasking
* 64×32 monochrome display
* Sprite rendering
* Keyboard input
* Delay and sound timers
* CHIP-8 font support

## Building

Requirements:

* Odin
* Raylib

```bash
odin build .
```

## Running

```bash
./chip8 [path to rom ]
```
